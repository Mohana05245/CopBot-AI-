# 🚔 CopBot AI – Multilingual Police FIR Drafting Assistant

An AI-powered multilingual FIR drafting assistant developed using Flask, NLP, and Machine Learning.

## ✨ Features

- AI Chatbot FIR Registration
- Crime Prediction using Machine Learning
- English & Tamil Language Support
- Automatic FIR PDF Generation
- Evidence Upload Support
- Modern Chatbot UI

## 🧠 Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Machine Learning
- NLP
- ReportLab

## 🤖 Machine Learning

Algorithm Used:
- Multinomial Naive Bayes

Purpose:
- Complaint Text Classification
- Crime Category Prediction
